2. Les 4 emplois
۴ کاربرد اصلی که باید بشناسی
😔 1. Regret — پشیمانی (sur le passé)
J'aurais dû vérifier les calculs avant le chantier.
باید قبل از کارگاه محاسبات را بررسی می‌کردم. (پشیمانم که نکردم)
Il aurait mieux valu choisir un autre emplacement.
بهتر بود مکان دیگری انتخاب می‌شد.
Nous aurions pu éviter ce problème structurel.
می‌توانستیم از این مشکل سازه‌ای جلوگیری کنیم.
😤 2. Reproche — سرزنش
Tu aurais pu me prévenir du retard !
می‌توانستی از تأخیر خبرم کنی! (سرزنش)
Vous auriez dû respecter les délais contractuels.
باید به مهلت‌های قراردادی احترام می‌گذاشتید.
🔑 3. Hypothèse sur le passé — فرض در گذشته (si + plus-que-parfait)
Si j'avais eu le budget, j'aurais choisi des matériaux premium.
اگر بودجه داشتم، مواد ممتاز انتخاب می‌کردم.
Si nous avions su, nous aurions modifié les plans.
اگر می‌دانستیم، پلان‌ها را تغییر می‌دادیم.
📰 4. Information non confirmée (passé) — خبر تأییدنشده
Selon le journal, l'architecte aurait remporté le concours.
به گفته روزنامه، ظاهراً معمار مسابقه را برده است.
3. Règle du SI — tableau complet
این جدول را حتماً حفظ کن!
Proposition SIProposition principaleSens
si + présentfutur simpleشرط واقعی — آینده
si + imparfaitconditionnel présentشرط غیرواقعی — حال/آینده
si + plus-que-parfaitconditionnel passéشرط غیرواقعی — گذشته
Si j'avais eu le temps, j'aurais relu le dossier. (PQP → cond. passé)
اگر وقت داشتم، پرونده را دوباره می‌خواندم. (در گذشته نداشتم)
⛔ La règle d'or : jamais «si + conditionnel» !
Si j'aurais su... ❌
Si j'avais su... ✅ (plus-que-parfait)
4. Niveau B1/B2 — Aller plus loin

🏗️ Contexte professionnel — Architecture

On aurait pu intégrer la géothermie dès la conception.
می‌توانستیم از همان مرحله طراحی، ژئوترمال را وارد کنیم.
Le client aurait préféré une façade en bois.
مشتری ترجیح می‌داد نمای چوبی داشته باشد.
Sans ce délai, le projet aurait été livré à temps.
بدون این تأخیر، پروژه به موقع تحویل می‌شد.

🔀 Structure mixte (B2) — si PQP → cond. présent

Si j'avais étudié l'ingénierie, je serais plus à l'aise avec les calculs.
اگر مهندسی خوانده بودم، الان با محاسبات راحت‌تر بودم. (گذشته → حال)
⭐ این ساختار در B2 خیلی مهم است!

1 Regret ou reproche ? Choisis la bonne forme

Si j'avais su, je ___ (ne pas accepter) ce projet.
اگر می‌دانستم، این پروژه را قبول نمی‌کردم.
n'aurais pas accepté — si + plus-que-parfait (avais su) → conditionnel passé
Tu ___ (devoir) vérifier les fondations avant de signer !
باید قبل از امضاء پی‌ها را بررسی می‌کردی! (سرزنش)
aurais dû — برای سرزنش در گذشته: conditionnel passé de «devoir»
Nous ___ (pouvoir) éviter cet échec si nous avions planifié mieux.
اگر بهتر برنامه‌ریزی می‌کردیم، می‌توانستیم از این شکست جلوگیری کنیم.
aurions pu — si + plus-que-parfait (avions planifié) → conditionnel passé
